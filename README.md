## Архитектура проекта

Проект следует принципам чистой архитектуры (Clean Architecture) в минимальном исполнении:

- **Модель (Model):** Классы, представляющие математические сущности (числа, операторы).
- **Логика (Core):** Калькулятор состояний, парсер выражений, обработка ошибок (деление на ноль и т.д.).
- **Пользовательский интерфейс (UI):** Отдельный слой, который может быть заменен (консоль, GUI, Web) без изменения логики калькулятора.

Подробнее см. в [`docs/architecture.md`](docs/architecture.md).
