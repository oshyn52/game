\# История изменений



Все заметные изменения в этом проекте будут задокументированы в этом файле.



Формат основан на \[Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),

и этот проект придерживается \[Semantic Versioning](https://semver.org/spec/v2.0.0.html).



\## \[1.1.0] - 2024-05-20



\### Добавлено

\-   Редис для кеширования популярных ссылок (уменьшение нагрузки на БД).

\-   Эндпоинт `/api/stats/{short\_code}` для получения статистики переходов.

\-   Инструкция по деплою в `docs/deployment.md`.



\### Изменено

\-   Обновлен образ Python с 3.9 до 3.11 в `Dockerfile`.

\-   Улучшена валидация URL (теперь поддерживаются схемы FTP).



\## \[1.0.0] - 2024-04-01



\### Добавлено

\-   Базовый функционал создания коротких ссылок.

\-   Редирект с короткой ссылки на оригинальный URL.

\-   Swagger документация (/docs).

\-   Docker-образ и docker-compose для локальной разработки.
