# Changelog

Все заметные изменения в этом проекте будут отражаться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased] — Дата неизвестна

### Добавлено
-   Страница истории вычислений с возможностью просмотра прошлых операций.
-   Кнопка "Сброс" (C) для очистки текущего ввода.

### Изменено
-   Главное меню приложения: иконки обновлены на более современный стиль.
-   Шрифт дисплея увеличен для лучшей читаемости.

### Исправлено
-   Исправлена ошибка, при которой деление на ноль приводило к вылету приложения (теперь показывает "Error").
-   Исправлено некорректное округление при вычислении процентов от дробных чисел.

## [1.2.0] — 2023-10-26

### Добавлено
-   Поддержка клавиатуры: теперь можно вводить цифры и операции, не нажимая на кнопки мышью.
-   Кнопка "Backspace" для удаления последнего введенного символа.
-   Тема "Тёмная" добавлена в настройки приложения.

### Изменено
-   Логика работы кнопки "+/-" (смена знака) теперь работает корректно для отрицательных чисел.

## [1.1.0] — 2023-09-15

### Добавлено
-   Реализованы кнопки памяти (M+, M-, MR, MC).
-   Добавлена поддержка операций с плавающей точкой (кнопка с точкой).

### Исправлено
-   Исправлена проблема с потерей точности при вычислениях с плавающей точкой (например, 0.1 + 0.2).
-   Мелкие исправления верстки для мобильных устройств.

## [1.0.0] — 2023-08-01

### Добавлено
-   Базовый графический интерфейс пользователя (кнопки и дисплей).
-   Основные арифметические операции: сложение, вычитание, умножение, деление.
-   Вывод результата на экран при нажатии кнопки "=".
