\# API и архитектура LinkCutter



\## Общая архитектура



Сервис построен по модульному принципу:

1\.  \*\*API Layer (FastAPI):\*\* Обрабатывает входящие запросы.

2\.  \*\*Service Layer:\*\* Содержит бизнес-логику (генерация кодов, валидация).

3\.  \*\*Repository Layer:\*\* Работа с базой данных (PostgreSQL).

4\.  \*\*Cache Layer:\*\* Redis для кеширования популярных ссылок.
